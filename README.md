# 📊 Amazon Prime Content Strategy Dashboard

## 🚀 Project Overview

This project analyzes Amazon Prime’s content library to uncover trends in content growth, distribution, and performance.
The goal is to provide actionable insights into content strategy and identify areas for improvement.

---

## 🎯 Objectives

* Analyze content growth over time
* Understand distribution between movies and shows
* Evaluate content performance using segmentation
* Build an interactive dashboard for business insights

---

## 🧹 Data Cleaning & Transformation

* Removed duplicates and inconsistencies from dataset
* Standardized categorical fields such as genres and content type
* Handled missing values for improved data quality
* Created a calculated field **Content Category**:

  * Blockbuster
  * Popular
  * Average
  * Low Performing

---

## 📊 Dashboard Features

* KPI Cards (Total Titles, Average IMDb Rating, Total Votes)
* Content Growth Trend (Year-wise analysis)
* Content Distribution (Movies vs Shows)
* Content Performance Segmentation
* Interactive dashboard actions (click-based filtering)

---

## 💡 Key Insights

* Movies dominate the platform (~85% of total content)
* Content growth accelerated significantly after 2010
* Shows are increasing steadily in recent years
* Majority of content falls under average or low-performing categories
* Very few titles qualify as high-performing (blockbusters)
